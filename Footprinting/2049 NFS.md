**NFS** is a system designed for **client/server** that enables users to seamlessly access files over a network as though these files were located within a local directory.

https://blog.1nf1n1ty.team/hacktricks/network-services-pentesting/nfs-service-pentesting

Enumeration
```bash
# Enumerate mounts
showmount -e <victim>  # make note of any shares

# nfs Commands
nfs-ls [-R] nfs://<victim>/<share>     # List contents, -R : recursive
nfs-cat nfs://<victim>/<share>/<file>  # Read contents
nfs-cp nfs://<victim>/<share>/<file> ./<destination>  # Copy contents
nfs-stat nfs://<victim>/<share>/<file> # Attributes of file
```
